# Random Quote Generator

**Difficulty:** Beginner

**Concepts:** Arrays/Lists, Random Selection, Indexing

---

## Description

Create a program that displays a random quote from a collection of quotes.

This challenge introduces:
- Storing multiple values in an array/list
- Generating random numbers
- Using random numbers to select from a list

---

## Requirements

- Store at least 5 quotes in your program
- When the program runs, randomly select one quote
- Display the selected quote

---

## Example

```
Here's a quote for you:

"The only way to do great work is to love what you do." - Steve Jobs
```

*(Running again might show:)*

```
Here's a quote for you:

"In the middle of difficulty lies opportunity." - Albert Einstein
```

## Hints

*There are many ways to solve this challenge. These hints may not apply to your language or approach.*

<details>
<summary>Hint 1</summary>

Create an array/list containing all your quotes as strings.

</details>

<details>
<summary>Hint 2</summary>

Generate a random number between 0 and the length of your list minus 1. Use this as the index to access a quote.

</details>

<details>
<summary>Hint 3</summary>

Most languages have a random module/library you need to import or include. Look up "random number" in your language.

</details>
