# What's Your Name?

**Difficulty:** Beginner

**Concepts:** Variables, User Input, String Concatenation, Output

---

## Description

Create a program that asks the user for their first, middle, and last name, then displays their full name.

---

## Requirements

- Ask the user for their first name
- Ask the user for their middle name
- Ask the user for their last name
- Display their complete name combined together

---

## Example

*Lines starting with `>` represent user input.*

```
What's your first name?
> Bob
And your middle name?
> O
And finally what's your last name?
> Henry
Cool, so your full name is Bob O Henry!
```

---

## Hints

*There are many ways to solve this challenge. These hints may not apply to your language or approach.*

<details>
<summary>Hint 1</summary>

Store each name part in a separate variable.

</details>

<details>
<summary>Hint 2</summary>

Combine the variables with spaces between them when outputting.

</details>

---

## Bonus Challenges

- [ ] Handle empty inputs gracefully (skip blank name parts)
- [ ] Capitalize the first letter of each name part
- [ ] Add a greeting using just their first name
