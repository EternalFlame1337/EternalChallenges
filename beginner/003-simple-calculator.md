# Simple Calculator

**Difficulty:** Beginner

**Concepts:** Variables, User Input, Basic Math Operations, Output

---

## Description

Create a program that asks the user for two numbers, then performs basic math operations on them and displays the results.

This challenge introduces you to:
- Getting number input from users
- Storing values in variables
- Using math operators (+, -, *, /)
- Displaying calculated results

---

## Requirements

- Ask the user to enter the first number
- Ask the user to enter the second number
- Calculate and display:
  - The sum (addition)
  - The difference (subtraction)
  - The product (multiplication)
  - The quotient (division)

---

## Example

*Lines starting with `>` represent user input.*

```
Enter the first number:
> 10
Enter the second number:
> 3

Results:
10 + 3 = 13
10 - 3 = 7
10 * 3 = 30
10 / 3 = 3.33
```

---

## Hints

*There are many ways to solve this challenge. These hints may not apply to your language or approach.*

<details>
<summary>Hint 1</summary>

Most languages require you to convert text input to a number before doing math. Look up how your language handles this (e.g., `int()` in Python, `parseInt()` in JavaScript).

</details>

<details>
<summary>Hint 2</summary>

Store each number in its own variable so you can use them multiple times in calculations.

</details>

<details>
<summary>Hint 3</summary>

Basic math operators work similarly in most languages: `+` for addition, `-` for subtraction, `*` for multiplication, `/` for division.

</details>
