# Multiplication Table

**Difficulty:** Beginner

**Concepts:** Loops, Variables, Math, Formatted Output

---

## Description

Create a program that displays the multiplication table for any number the user chooses.

This challenge practices:
- Using loops to repeat calculations
- Combining loop variables with other values
- Formatting output to look clean

---

## Requirements

- Ask the user for a number
- Display the multiplication table for that number (1 through 10)
- Format the output clearly

---

## Example

*Lines starting with `>` represent user input.*

```
Enter a number:
> 7

7 x 1 = 7
7 x 2 = 14
7 x 3 = 21
7 x 4 = 28
7 x 5 = 35
7 x 6 = 42
7 x 7 = 49
7 x 8 = 56
7 x 9 = 63
7 x 10 = 70
```

---

## Hints

*There are many ways to solve this challenge. These hints may not apply to your language or approach.*

<details>
<summary>Hint 1</summary>

Use a loop that counts from 1 to 10. The loop counter is what you multiply by.

</details>

<details>
<summary>Hint 2</summary>

Inside the loop, multiply the user's number by the loop counter and display the result.

</details>
