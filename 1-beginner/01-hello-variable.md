# Hello Variable

**Difficulty:** Beginner

**Concepts:** Variables, String Concatenation, Output

---

## Description

Create a program that stores a value in a variable, then prints that value followed by a greeting message.

---

## Requirements

- Create a variable and assign it any value you choose
- Print the variable's value followed by " Hello!"
- The greeting must be added during output, not stored in the original variable

---

## Example

```
OKIDADEEERAAD Hello!
```

In this example, the variable contained `"OKIDADEEERAAD"` and the program appended `" Hello!"` when printing.

---

## Hints

*There are many ways to solve this challenge. These hints may not apply to your language or approach.*

<details>
<summary>Hint 1</summary>

Most languages let you combine strings with a `+` operator or similar syntax.

</details>

<details>
<summary>Hint 2</summary>

Make sure there's a space before "Hello!" so the output looks clean.

</details>
